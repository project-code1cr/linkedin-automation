# LinkedIn Automation Assignment

## Overview
This project is a backend-driven LinkedIn automation prototype built using Node.js and Playwright. 
It demonstrates how LinkedIn actions can be automated safely while applying basic anti-bot strategies.

## Features
- Backend API to trigger automation
- Manual LinkedIn login using real browser
- Automated visiting of multiple LinkedIn profiles
- Random human-like delays to avoid bot detection

## Tech Stack
- Node.js
- Express.js
- Playwright

## Anti-Bot Strategies Used
- Real browser (headless disabled)
- Manual login
- Sequential actions
- Random delays between actions
- Limited scope automation

## How to Run
1. Install dependencies: `npm install`
2. Start server: `node server/index.js`
3. Open browser: `http://localhost:3000/start`
